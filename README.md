# FindHabitat
The project is named FindHabitat. 

Currently due to severe nationwide housing crisis and extreme high rental prices, it is hard to find an adequate house. Especially in metropolitan areas. A lot of competition over good houses makes a great challenge especially for students who try to find house after being accepted into a university. 

The aim is to build a web application that supports a user to post a home and other users to find their ideal house.
